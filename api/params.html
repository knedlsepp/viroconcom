__init__(a, b, c, func_type, wrapper=None)[source]¶
Parameters:
  • a,b,c (float,) – The function parameters.
  • func_type (str,) –
    Defines which kind of function to use:
    f1:\(a + b * x^c\)
    f2:\(a + b * e^{x * c}\)
  • wrapper (function or Wrapper,) – A function or a Wrapper object to wrap around the function. The function has to be pickleable. (i.e. lambdas, clojures, etc. are not supported.) Using this wrapper, one can e.g. create \(exp(a + b * x^c)\) with func_type=polynomial and wrapper=math.exp.
